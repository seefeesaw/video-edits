Space-optimised alternative
Keep only a, b, c.
Each step: a, b, c = b, c, a+b+c
Step 6 — Complete Solution

O(n) time  ·  O(1) space

def tribonacci(n):
  if n == 0: return 0
  if n <= 2: return 1
  a, b, c = 0, 1, 1
  for _ in range(n − 2):
    a, b, c = b, c, a + b + c
  return c
