0/1 Knapsack

You have a knapsack with a fixed weight capacity. Each item has a weight and a value. Select items to maximize total value — without exceeding capacity. Each item is either fully included or excluded; no fractions.

1 ≤ capacity ≤ 1000 1 ≤ values.length ≤ 500 weights.length == values.length

The key insight — take it or leave it

One binary choice per item,
every subproblem captured once

Define dp[i][w] = best value using the first i items within weight limit w. Each cell answers: for this item, take it or skip it?

dp[0][w] = 0                  # base: no items
 
dp[i][w] = dp[i-1][w]            # skip (can't fit)
           if weights[i] > w
 
dp[i][w] = max(dp[i-1][w],       # skip
                values[i] + dp[i-1][w - weights[i]])
           # take it

Python · Complete Solution

The full knapsack() function

from typing import List
 
def knapsack(capacity: int, weights: List[int], values: List[int]) -> int:
    n = len(weights)
    dp = [[0] * (capacity + 1) for _ in range(n + 1)]
 
    for i in range(1, n + 1):
        w_i = weights[i - 1]
        v_i = values[i - 1]
        for w in range(capacity + 1):
            dp[i][w] = dp[i - 1][w]
            if w_i <= w:
                dp[i][w] = max(dp[i][w], v_i + dp[i - 1][w - w_i])
 
    return dp[n][capacity]
